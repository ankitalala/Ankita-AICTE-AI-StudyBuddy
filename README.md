🧠 AI-Powered Study Buddy (Study Dost)

An intelligent AI-based study assistant that helps students learn efficiently by explaining concepts, summarizing notes, and generating quizzes using Natural Language Processing (NLP).

🚀 Features
📚 Explain complex topics in simple language
🧾 Summarize long study materials
🎯 Generate quiz questions for practice
⚡ Real-time interaction using a web interface


🛠️ Technologies Used
Python
Hugging Face Transformers (BART, FLAN-T5)
Streamlit
Google Colab
Pyngrok
JSON


🧠 Models Used
facebook/bart-large-cnn → Text Summarization
google/flan-t5 → Explanation & Quiz Generation
